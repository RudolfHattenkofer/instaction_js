Simple example

This input will be replaced with its value:

<input type="text" class="readonly-input" value="Visible on page load">

Instaction.register('.readonly-input', function( $node ) {
    $node.replaceWith($node.val());
});

jQuery selector

    $node.after($node.data('tooltip'));
